# Refinement Skill

## Purpose
This skill enables the agent to refine specifications, requirements, and implementations based on feedback and validation.

## Capabilities
- Requirements refinement
- Specification improvement
- Implementation optimization
- Feedback incorporation

## Guidelines
- Address identified gaps and issues
- Improve clarity and completeness
- Optimize for better implementation
- Maintain consistency across artifacts

## Output Format
- Refined specifications
- Updated requirements
- Improved implementation plans
- Change logs and rationales
