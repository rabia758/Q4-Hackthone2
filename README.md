        # Q4-Hackthone-2

This is a multi-phase project implementing various applications:

- **Phase 1**: Console-based Todo Application
- **Phase 2**: Full-Stack Web Todo Application
- **Phase 3**: Chatbot
- **Phase 4**: Local K8s Deployment
- **Phase 5**: Console App Enhancement

## Phase 2: Full-Stack Web Todo Application

This phase implements a modern, full-stack web application with:

- Next.js frontend with App Router
- FastAPI backend with SQLModel ORM
- Neon PostgreSQL database
- JWT-based authentication
- Responsive UI with Tailwind CSS

### Tech Stack
- Frontend: NextJS, TypeScript, Tailwind CSS
- Backend: FastAPI, SQLModel, PostgreSQL
- Authentication: JWT with proper password hashing
- Database: Neon Serverless PostgreSQL

### Features
- User authentication (signup/signin)
- Todo management (CRUD operations)
- Responsive design following modern minimalist UI principles
- Secure password handling
- RESTful API design

For detailed setup instructions, see the README in the `phase-2-web-app` directory.
